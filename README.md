# Midwest Conservation Blueprint Explorer

This application provides a simple user interface for exploring the
[Midwest Conservation Blueprint](https://mcap-fws.hub.arcgis.com/pages/midwest-conservation-blueprint).

This project is adapted from the
[Southeast Conservation Blueprint Explorer](https://github.com/astutespruce/secas-blueprint).

## Local development

See [local development instructions](./Developing.md)

## Server Deployment

This application is co-hosted with the
[Southeast Conservation Blueprint Explorer](https://github.com/astutespruce/secas-blueprint)
on USFWS Azure servers, according to the configuration available
[here](https://github.com/astutespruce/secas-docker).
